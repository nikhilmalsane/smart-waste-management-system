### \#Smart Waste Management System



A MERN stack based web application to manage the waste bin collection using a centralized dashboard.



##### 

###### Objective :- 

&nbsp;		The main goal of this project is to provide a digital solution for tracking waste bins, improving waste collection efficiency, and maintaining cleanliness in smart cities.





###### Tech-Stack :- 

&nbsp;		MongoDB (Database)

&nbsp;		Express.js (Backend framework)

&nbsp;		React.js (Frontend library)

&nbsp;		Node.js (Runtime Environment)
